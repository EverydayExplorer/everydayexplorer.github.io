Implementing Data Structures in Java: Linked Lists


Linked lists are one of the fundamental data structures in computer science. They provide a flexible way of storing and manipulating data, making them essential in many programming applications. In this blog post, we will explore how to implement linked lists in Java and discuss their advantages and use cases.

What is a Linked List?

A linked list is a collection of nodes, where each node contains a value and a reference to the next node in the list. Unlike arrays, linked lists do not require contiguous memory allocation. This flexibility allows for efficient insertion and deletion operations, as well as dynamic resizing.

Implementing a Singly Linked List

To implement a linked list in Java, we start by defining a Node class, which represents each node in the list. Each node contains a value and a reference to the next node.

class Node {
    int value;
    Node next;
    
    Node(int value) {
        this.value = value;
        this.next = null;
    }
}

Next, we define the LinkedList class, which acts as a wrapper around the nodes. It keeps track of the head node, which is the entry point to the list.

class LinkedList {
    Node head;
    
    LinkedList() {
        this.head = null;
    }
    
    // Other methods will be discussed later
}

Now, let’s explore some common operations that can be performed on a linked list.

Insertion

To insert a new node at the beginning of the list, we update the head reference to point to the new node, and set the next reference of the new node to the previous head.

void insertAtBeginning(int value) {
    Node newNode = new Node(value);
    newNode.next = head;
    head = newNode;
}

To insert a new node at the end of the list, we traverse the list until we reach the last node, and update its next reference to point to the new node.

void insertAtEnd(int value) {
    Node newNode = new Node(value);
    
    if (head == null) {
        head = newNode;
    } else {
        Node current = head;
        while (current.next != null) {
            current = current.next;
        }
        current.next = newNode;
    }
}

Deletion

To delete a node from the list, we need to update the references of the previous and next nodes. If the node to be deleted is the head node, we simply update the head reference to the next node. Otherwise, we traverse the list to find the node to be deleted and update the references accordingly.

void deleteNode(int value) {
    if (head == null) {
        return;
    }
    
    if (head.value == value) {
        head = head.next;
        return;
    }
    
    Node current = head;
    Node previous = null;
    
    while (current != null && current.value != value) {
        previous = current;
        current = current.next;
    }
    
    if (current == null) {
        return;
    }
    
    previous.next = current.next;
}

Traversal

To traverse a linked list and print its elements, we start from the head node and move to the next node until we reach the end of the list.

void printList() {
    Node current = head;
    
    while (current != null) {
        System.out.println(current.value);
        current = current.next;
    }
}

Conclusion

In this blog post, we explored the implementation of linked lists in Java. Linked lists offer flexibility and efficiency for various operations, such as insertion, deletion, and traversal. They are widely used in many programming applications, including data processing, graph algorithms, and dynamic memory allocation.

By understanding the fundamentals of linked lists and their implementation in Java, you can enhance your programming skills and leverage this powerful data structure to solve complex problems efficiently. Happy coding!
